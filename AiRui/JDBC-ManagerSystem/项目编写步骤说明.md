# 商店库存管理系统 - 项目编写步骤说明

## 1. 数据库设计

### 1.1 数据库表结构

| 表名 | 功能描述 | 字段说明 |
|------|---------|----------|
| `user` | 用户管理 | user_id (主键), username (用户名), password (密码), real_name (真实姓名), role (角色), status (状态), create_time (创建时间), update_time (更新时间) |
| `supplier` | 供应商管理 | supplier_id (主键), supplier_name (供应商名称), contact_person (联系人), phone (联系电话), address (地址), status (状态), create_time (创建时间), update_time (更新时间) |
| `product` | 产品管理 | product_id (主键), product_name (产品名称), category (分类), unit (单位), price (单价), supplier_id (外键), status (状态), create_time (创建时间), update_time (更新时间) |
| `inventory` | 库存管理 | inventory_id (主键), product_id (外键), quantity (库存数量), min_stock (最低库存预警), last_update_time (最后更新时间) |
| `inventory_transaction` | 库存交易管理 | transaction_id (主键), product_id (外键), transaction_type (交易类型), quantity (数量), unit_price (单价), total_amount (总金额), operator_id (操作人ID), transaction_time (交易时间), remark (备注) |

### 1.2 数据库关系
- `user` 表与 `inventory_transaction` 表：一对多关系，一个用户可以执行多条交易记录
- `supplier` 表与 `product` 表：一对多关系，一个供应商可以提供多种产品
- `product` 表与 `inventory` 表：一对一关系，一个产品对应一条库存记录
- `product` 表与 `inventory_transaction` 表：一对多关系，一个产品可以有多条交易记录

## 4. 项目分层架构

### 4.1 分层结构

| 层级 | 职责 | 主要包 |
|------|------|--------|
| 表示层 (Controller) | 处理用户输入输出，显示菜单，调用服务层方法 | `controller` |
| 业务逻辑层 (Service) | 实现业务逻辑，处理事务，调用DAO层方法 | `service` |
| 数据访问层 (DAO) | 负责数据库操作，执行SQL语句 | `dao` |
| 数据模型层 (Model) | 定义数据结构，对应数据库表 | `model` |
| 工具层 (Util) | 提供通用工具方法 | `util` |

### 4.2 分层职责详解

#### 表示层 (Controller)
- 负责与用户交互，显示菜单和接收用户输入
- 调用业务逻辑层的方法处理业务需求
- 处理异常并向用户显示错误信息
- 控制流程跳转

#### 业务逻辑层 (Service)
- 实现核心业务逻辑
- 处理事务管理，确保数据一致性
- 调用DAO层方法进行数据操作
- 验证业务规则和数据有效性

#### 数据访问层 (DAO)
- 负责与数据库交互
- 执行SQL语句，处理结果集
- 提供基本的CRUD操作
- 封装数据库操作细节

#### 数据模型层 (Model)
- 定义数据结构，对应数据库表
- 提供getter和setter方法
- 实现业务实体的属性和行为

#### 工具层 (Util)
- 提供通用工具方法
- 处理数据库连接、输入验证、密码加密等功能
- 封装重复使用的代码

## 5. 功能实现流程

### 5.1 用户登录功能

**流程说明**：
1. 用户在控制台输入用户名和密码
2. 系统验证输入的有效性
3. 系统调用UserService的login方法进行登录验证
4. UserService调用UserDAO查询用户信息
5. 系统验证密码是否正确
6. 系统根据验证结果显示相应信息

**核心代码实现**：

```java
// MenuController.java - login方法
private void login() {
    InputUtil.clearScreen();
    System.out.println("========================================");
    System.out.println("       商店库存管理系统 - 登录");
    System.out.println("========================================");

    String username = InputUtil.readString("请输入用户名: ");
    String password = InputUtil.readString("请输入密码: ");

    try {
        currentUser = userService.login(username, password);
        if (currentUser != null) {
            System.out.println("登录成功！欢迎回来，" + currentUser.getRealName() + "！");
            InputUtil.pause();
        } else {
            System.out.println("用户名或密码错误，请重试！");
            InputUtil.pause();
        }
    } catch (SQLException e) {
        System.out.println("登录失败: " + e.getMessage());
        InputUtil.pause();
    }
}

// UserService.java - login方法
public User login(String username, String password) throws SQLException {
    Connection conn = null;
    try {
        conn = getConnection();
        User user = userDAO.findByUsername(conn, username);
        if (user != null && user.getStatus() == 1 && PasswordUtil.checkPassword(password, user.getPassword())) {
            return user;
        }
        return null;
    } finally {
        closeConnection(conn);
    }
}

// UserDAO.java - findByUsername方法
public User findByUsername(Connection conn, String username) throws SQLException {
    String sql = "SELECT * FROM user WHERE username = ?";
    return queryOne(conn, sql, this::mapRow, username);
}
```

### 5.2 产品管理功能

**流程说明**：
1. 用户选择产品管理菜单

4. 系统提示用户输入产品信息

7. ProductService调用ProductDAO插入产品数据
8. 系统显示操作结果

**核心代码实现**：

```java
// ProductController.java - addProduct方法
private void addProduct() {
    InputUtil.clearScreen();
    System.out.println("========================================");
    System.out.println("       商店库存管理系统 - 添加产品");
    System.out.println("========================================");

    String productName = InputUtil.readString("请输入产品名称: ");
    String category = InputUtil.readString("请输入产品分类: ");
    String unit = InputUtil.readString("请输入计量单位: ");
    double price = InputUtil.readDouble("请输入单价: ");
    int supplierId = InputUtil.readInt("请输入供应商ID: ");

    // 验证输入
    if (!ValidationUtil.isValidPrice(price)) {
        System.out.println("价格格式不正确！");
        InputUtil.pause();
        return;
    }

    try {
        // 检查供应商是否存在
        Supplier supplier = supplierService.getSupplierById(supplierId);
        if (supplier == null) {
            System.out.println("供应商不存在！");
            InputUtil.pause();
            return;
        }

        Product product = new Product();
        product.setProductName(productName);
        product.setCategory(category);
        product.setUnit(unit);
        product.setPrice(new BigDecimal(price));
        product.setSupplierId(supplierId);
        product.setStatus(1);

        boolean result = productService.addProduct(product);
        if (result) {
            System.out.println("添加产品成功！");
        } else {
            System.out.println("添加产品失败！");
        }
    } catch (SQLException e) {
        System.out.println("添加产品失败: " + e.getMessage());
    }
    InputUtil.pause();
}

// ProductService.java - addProduct方法
public boolean addProduct(Product product) throws SQLException {
    Connection conn = null;
    try {
        conn = getConnection();
        int result = productDAO.insert(conn, product);
        return result > 0;
    } finally {
        closeConnection(conn);
    }
}

// ProductDAO.java - insert方法
public int insert(Connection conn, Product product) throws SQLException {
    String sql = "INSERT INTO product (product_name, category, unit, price, supplier_id, status) VALUES (?, ?, ?, ?, ?, ?)";
    return executeUpdate(conn, sql, product.getProductName(), product.getCategory(), product.getUnit(), product.getPrice(), product.getSupplierId(), product.getStatus());
```

### 5.3 库存入库功能

**流程说明**：
1. 用户选择库存管理菜单
2. 系统显示库存管理子菜单
3. 用户选择入库操作
4. 系统提示用户输入入库信息
5. 系统验证输入的有效性
6. 系统调用InventoryService的addStock方法进行入库操作
7. InventoryService开启事务，更新库存并记录交易
8. 系统显示操作结果

**核心代码实现**：

```java
// InventoryController.java - addStock方法
private void addStock(User currentUser) {
    InputUtil.clearScreen();
    System.out.println("========================================");
    System.out.println("       商店库存管理系统 - 入库操作");
    System.out.println("========================================");

    int productId = InputUtil.readInt("请输入产品ID: ");
    int quantity = InputUtil.readInt("请输入入库数量: ");
    double unitPrice = InputUtil.readDouble("请输入入库单价: ");
    String remark = InputUtil.readString("请输入备注: ");

    // 验证输入
    if (!ValidationUtil.isValidQuantity(quantity) || !ValidationUtil.isValidPrice(unitPrice)) {
        System.out.println("数量或价格格式不正确！");
        InputUtil.pause();
        return;
    }

    try {
        // 检查产品是否存在
        Product product = productService.getProductById(productId);
        if (product == null) {
            System.out.println("产品不存在！");
            InputUtil.pause();
            return;
        }

        // 检查库存是否存在
        Inventory inventory = inventoryService.getInventoryByProductId(productId);
        if (inventory == null) {
            System.out.println("该产品库存不存在，请先创建库存记录！");
            InputUtil.pause();
            return;
        }

        boolean result = inventoryService.addStock(productId, quantity, new BigDecimal(unitPrice), currentUser.getId(), remark);
        if (result) {
            System.out.println("入库成功！");
        } else {
            System.out.println("入库失败！");
        }
    } catch (SQLException e) {
        System.out.println("入库失败: " + e.getMessage());
    }
    InputUtil.pause();
}

// InventoryService.java - addStock方法
public boolean addStock(Integer productId, Integer quantity, BigDecimal unitPrice, Integer operatorId, String remark) throws SQLException {
    Connection conn = null;
    try {
        conn = getConnection();
        beginTransaction(conn);
        
        // 更新库存
        inventoryDAO.updateQuantity(conn, productId, quantity);
        
        // 记录交易
        InventoryTransaction transaction = new InventoryTransaction();
        transaction.setProductId(productId);
        transaction.setTransactionType("in");
        transaction.setQuantity(quantity);
        transaction.setUnitPrice(unitPrice);
        transaction.setTotalAmount(unitPrice.multiply(new BigDecimal(quantity)));
        transaction.setOperatorId(operatorId);
        transaction.setRemark(remark);
        transactionDAO.insert(conn, transaction);
        
        commitTransaction(conn);
        return true;
    } catch (SQLException e) {
        rollbackTransaction(conn);
        throw e;
    } finally {
        closeConnection(conn);
    }
}
```

在实现过程中，我们注重代码的可读性、可维护性和安全性，采用了事务管理确保数据一致性，使用异常处理提高系统稳定性，通过输入验证增强系统安全性。
